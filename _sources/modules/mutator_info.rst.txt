mutator_info -- List Sensu mutators
===================================

Retrieve information about Sensu mutators.

For more information, refer to the Sensu documentation at https://docs.sensu.io/sensu-go/latest/reference/mutators/.


Examples
--------

.. code-block:: yaml+jinja

   
   - name: List all Sensu mutators
     mutator_info:
     register: result

   - name: Retrieve a single Sensu mutator
     mutator_info:
       name: my-mutator
     register: result





See Also
--------

.. seealso::

   - :ref:`mutator_module`



Parameters
----------

auth (optional)
  Authentication parameters. Can define each of them with ENV as well.

  | **type**: dict

  password (optional)
    The Sensu user's password. If this is not set the value of the SENSU_PASSWORD environment variable will be checked.

    | **type**: str
    | **default**: P@ssw0rd!

  url (optional)
    Location of the Sensu backend API. If this is not set the value of the SENSU_URL environment variable will be checked.

    | **type**: str
    | **default**: http://localhost:8080

  user (optional)
    The username to use for connecting to the Sensu API. If this is not set the value of the SENSU_USER environment variable will be checked.

    | **type**: str
    | **default**: admin


name (optional)
  Retrieve information about this specific object instead of listing all objects.

  | **type**: str

namespace (optional)
  RBAC namespace to operate in. If this is not set the value of the SENSU_NAMESPACE environment variable will be used.

  | **type**: str
  | **default**: default





Return Values
-------------

objects (always), list, )
  list of Sensu mutators
